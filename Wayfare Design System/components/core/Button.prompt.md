Primary action control — use for any commitment in a flow (enquire, book, search); pair one `primary` (coral) with quieter `outline`/`ghost` siblings.

```jsx
<Button variant="primary" size="lg">Plan my trip</Button>
<Button variant="outline">Explore trips</Button>
<Button variant="link">Read the story</Button>
```

Variants: `primary` (coral fill — one per view), `dark` (ink fill, for use on cream imagery), `outline` (ink hairline → fills ink on hover), `ghost` (low-emphasis), `link` (uppercase tracked underline, editorial inline CTA). Sizes `sm | md | lg`. Props: `fullWidth`, `iconLeft`, `iconRight`, `as="a"` for links.
